# **App Name**: CRApro95

## Core Features:

- Dashboard: Display key metrics: total plots, total events, recent events.
- Plots Management: Create, read, update, and delete plots with details such as name, code, area, location, and status.
- Crops Management: Manage crops, including names and descriptions.
- Seasons Management: Handle seasons (zafras) including creation, editing, and closing.
- Event Logging: Log agricultural events with details like type, date, description, inputs, quantity, and results. Events are linked to plots and seasons.
- User Role Management: Manage user accounts with roles (admin, operator, consult) and permissions.
- Role Management: Create, read, update, and delete roles with descriptions.
- Firestore Integration: Utilize Firestore collections: plots, crops, seasons, events, users, and roles, complete with necessary read/write rules for authentication and authorization.

## Style Guidelines:

- Primary color: Earthy green (#6B8E23) to evoke agriculture and nature.
- Background color: Light beige (#F5F5DC) to provide a neutral, calming backdrop.
- Accent color: Warm orange (#D2691E) for highlights and calls to action.
- Body font: 'PT Sans', sans-serif for a modern and readable interface.
- Headline font: 'Alegreya', serif, to give an elegant and contemporary feel.
- Use clear and symbolic icons for navigation and data visualization.
- Design a clear layout with a side navigation bar.
